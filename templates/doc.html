post /api/inspection

Insérer dans la base de données une demande d'inspection.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • nom: required(string)
  • adresse: required(string)
  • ville: required(string)
  • date_visite: required(string)
  • nom_plaignant: required(string)
  • description: required(string)

Example:

{
  "nom": "Basha",
  "adresse": "265 St Catherine St E, H2X 1L5",
  "ville": "Montréal",
  "date_visite": "2019-04-17",
  "nom_plaignant": "Ouss",
  "description": "Il y avait des rats partout."
}

HTTP status code 201

La personne a bien été rajoutée.

HTTP status code 400

Les données reçues ne sont pas valide.

delete /api/inspection

Supprimer une demande d'inspection, spécifié par un id.

  • Response

HTTP status code 200

La personne a bien été supprimé.

HTTP status code 400

/api/contrevenants get

get

Renvoie les contrevenants qui ont eu des infractions entre deux dates spécifiées.

get /api/contrevenants

Renvoie les contrevenants qui ont eu des infractions entre deux dates spécifiées.

  • Request
  • Response

Query Parameters

  • du: required(string)

    Date de début de la recherche en format YYYY-DD-MM.

    Example:

    2019-11-11
  • au: required(string)

    Date de fin de la recherche en format YYYY-DD-MM.

    Example:

    2019-11-11

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: ExistingContrevenant

  • id: required(number)
  • proprietaire: required(string)
  • categorie: required(string)
  • etablissement: required(string)
  • adresse: required(string)
  • ville: required(string)
  • contraventions: required(array of ExistingContravention)

    Items: ExistingContravention

    • id: required(number)
    • proprietaire: required(string)
    • description: required(string)
    • date_infraction: required(string)
    • date_jugement: required(string)
    • montant: required(number)
